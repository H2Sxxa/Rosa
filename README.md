# 何为Rosa

Rosa是一款用于加速构建的工具箱，前身为MCreatorSetup，使用Dart重构并改为GUI应用

# 功能速览

 - 一键配置Proxifier（使用土球的代理服务器）
 - 修补ForgeGradle（替换为mcbbs的maven与assets）
 - mclogs粘贴板
 - MCreator专用配置JDK
